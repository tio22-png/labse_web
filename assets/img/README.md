# Folder untuk menyimpan gambar

## Logo Politeknik Negeri Malang

Download logo dari:
https://www.polinema.ac.id/

Atau gunakan logo yang sudah Anda punya, lalu:
1. Save dengan nama: `logo-pnm.png`
2. Letakkan di folder ini: `assets/img/logo-pnm.png`
3. Ukuran rekomendasi: 150x150px atau 200x200px (PNG dengan background transparan)

## Format yang Didukung
- PNG (rekomendasi - dengan background transparan)
- JPG
- SVG
